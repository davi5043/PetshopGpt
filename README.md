# 🐾 PetLux SaaS - Pet Care Platform

Plataforma SaaS moderna para pet shop premium com agendamentos, loja de produtos e dashboard interativo.

---

## 🚀 Sobre o projeto

O PetLux SaaS é uma aplicação front-end estilo startup real, inspirada em dashboards modernos como Stripe, Shopify e Notion.

Ele simula um sistema completo de pet shop digital com experiência de usuário profissional:

- 🗓️ Agendamento de serviços para pets  
- 🛒 Loja virtual de produtos pet  
- 📊 Dashboard administrativo  
- 🔔 Sistema de notificações (toast)  
- 💾 Persistência com LocalStorage  
- 🎨 Interface moderna estilo SaaS (dark mode)

---

## 🧠 Tecnologias utilizadas

- HTML5  
- CSS3 (Design System SaaS)  
- JavaScript (Vanilla JS)  
- LocalStorage  

---

## ⚙️ Funcionalidades

### 📊 Dashboard
- Total de agendamentos
- Carrinho de compras
- Resumo de pedidos

### 🐶 Serviços
- Banho Premium
- Banho + Tosa
- Agendamento completo

### 🛒 Loja
- Produtos pet com imagens
- Carrinho funcional
- Cálculo automático de total

### 📅 Agendamentos
- Nome do pet
- Data e horário
- Lista dinâmica em tempo real

### 🔔 UX
- Notificações tipo toast
- Navegação estilo SPA (sem reload)
- Interface responsiva

## ▶️ Como executar o projeto

Clone o repositório:

git clone https://github.com/davi5043/PetshopGpt.git

Entre na pasta:

cd petlux-saas

Abra o arquivo:

index.html

Ou use a extensão Live Server no VSCode.

---

## 🌐 Deploy no GitHub Pages

1. Vá em Settings  
2. Clique em Pages  
3. Selecione branch: main / root  
4. Clique em Save  

Seu projeto ficará online automaticamente.
